# Промежуточные данные (raw / intermediate)

Эти файлы – промежуточные артефакты пайплайна сбора данных. **Дашборду они не нужны** – он читает только `../city_registry.json` и `../novosibirsk_streets.json`. Они хранятся для прозрачности и воспроизводимости исследования.

| Файл | Описание |
|---|---|
| `novosibirsk_osm_all.json` | Сырая выгрузка всех улиц/топонимов Новосибирска из OpenStreetMap (Overpass API). |
| `novosibirsk_wikidata_p138.json` | Связки «улица → именованная сущность» (свойство P138 «named after») из Wikidata. |
| `novosibirsk_unverified.csv` | Топонимы, помеченные как НЕ именные (обычные городские названия) с пояснениями. |
| `found_in_wikipedia.csv` | Кандидаты-персоны, подтверждённые через Википедию. |
| `potential_named.csv` | Кандидаты в именные улицы для ручной проверки. |

Итоговый, очищенный и обогащённый набор – `../novosibirsk_streets.json` в родительской папке.
