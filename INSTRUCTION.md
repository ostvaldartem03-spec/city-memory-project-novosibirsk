# Как запустить дашборд – пошаговая инструкция

Это интерактивная карта имён на улицах Новосибирска: в честь кого названы улицы – героев войны, учёных, писателей, художников, космонавтов и других известных людей.

> Дашборд нужно открывать через локальный сервер (ниже). Прямое открытие файла `index.html` через `file://` не загрузит данные. Также нужен интернет – карта, библиотеки и часть портретов подгружаются онлайн.

## Шаг 1. Скачайте проект

Клонируйте репозиторий или скачайте и распакуйте ZIP-архив с GitHub (кнопка **Code → Download ZIP**), запомните папку.

## Шаг 2. Установите Python (если ещё нет)

1. Откройте https://www.python.org/downloads/ и скачайте последнюю версию.
2. При установке (Windows) обязательно поставьте галочку **«Add Python to PATH»**.

Python нужен только как простой локальный веб-сервер. Если у вас уже есть другой способ поднять статический сервер (Node, VS Code Live Server и т.п.) – можно использовать его.

## Шаг 3. Запустите локальный сервер

1. Откройте терминал в корневой папке проекта.
   - **Windows:** зажмите Shift + правая кнопка мыши → «Открыть окно PowerShell здесь».
   - **macOS/Linux:** откройте Terminal в этой папке.
2. Выполните команду:

   ```
   python -m http.server 8080
   ```
   (в Windows, если `python` не работает, попробуйте `py -m http.server 8080`)

3. Появится сообщение вроде `Serving HTTP on 0.0.0.0 port 8080 ...`. **Не закрывайте это окно** – пока оно открыто, сайт работает.

## Шаг 4. Откройте сайт в браузере

Откройте `http://localhost:8080/dashboard/` – появится карта Новосибирска с цветными точками.

## Что можно делать на сайте

- **Карта** – кликайте по точкам, чтобы узнать, в честь кого названа улица.
- **Фильтры** – по полу, профессии или эпохе.
- **Сеть персоналий** – граф связей между людьми и сферами деятельности.
- **Статистика** – диаграммы по полу, профессиям, эпохам.
- **🌓 Кнопка темы** – переключение тёмной/светлой темы.
- **Поиск** – по улицам или людям.

## Как остановить сайт

В окне терминала нажмите **Ctrl + C**, затем можно закрыть окно.

## Если не работает

- **«python / py не найден»** – Python не установлен или не отмечена галочка «Add Python to PATH». Переустановите.
- **Сайт не открывается** – проверьте, что окно сервера открыто; попробуйте `http://127.0.0.1:8080/dashboard/`.
- **Порт 8080 занят** – запустите на другом порту: `python -m http.server 9090`, тогда откройте `http://localhost:9090/dashboard/`.
- **Долго грузится** – подождите 5–10 секунд при первом открытии.
